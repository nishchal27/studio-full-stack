# Implementation Plan

## Phase 0: Foundation

- Create Next.js App Router setup with strict TypeScript.
- Establish folder structure and domain contracts.
- Add Zod schemas for page and section validation.
- Add typed section registry with fallback rendering.
- Add Contentful adapter contract methods.
- Add Redux Toolkit store and baseline slices.
- Add placeholder preview and studio routes.
- Add Playwright, axe, lint, typecheck, and CI scaffolding.

## Later Phases

- Phase 1: Contentful integration and schema renderer.
- Phase 2: Studio editor, RBAC, and draft persistence.
- Phase 3: publishing engine, SemVer diffing, release snapshots, and quality enforcement.
